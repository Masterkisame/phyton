Este repositorio contiene varios proyectos desarrollados en Python, organizados en carpetas para facilitar su exploración.

Contenido:
- Carreras/        → Juegos de carreras simples en Pygame
- Snake/           → Variantes del clásico juego de la serpiente (con selector, sonido, sprites, fondo)
- barman/          → Juego de simulación de bar con HUD y sistema de proveedores
- breakout/        → Versión del clásico juego de romper bloques
- carrera de ratas/→ Inspirado en Rodent’s Revenge y juegos retro
- fruit down/      → Juego arcade de frutas cayendo
- memoria/         → Juego de cartas para ejercitar la memoria
- mercado/         → Sistema de inventario y mercado en Pygame
- puzzle/          → Rompecabezas interactivo en Pygame
- ruleta/          → Juego de azar con mecánicas simples

Cómo ejecutar:
Cada carpeta tiene un archivo principal (main.py o similar).
Ejemplo:
python Snake/snake.py

Requisitos:
- Python 3.10 o superior
- Librería Pygame (instalar con: pip install pygame)
